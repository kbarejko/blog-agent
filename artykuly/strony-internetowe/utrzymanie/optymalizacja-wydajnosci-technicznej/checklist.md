## Checklist: Optymalizacja Wydajności Technicznej Strony

- [ ] Przetestuj obecną szybkość witryny w Google PageSpeed Insights i zachowaj wyniki jako bazę do porównań
- [ ] Przejrzyj rozmiar grafik na stronie głównej - warto skompresować te powyżej 100KB używając TinyPNG lub podobnego narzędzia
- [ ] Wdróż lazy loading dla obrazów poprzez dedykowaną wtyczkę lub funkcję dostępną w Twoim motywie
- [ ] Dokonaj przeglądu zainstalowanych wtyczek - usuń nieaktywne i rozważ zastąpienie zasobożernych lżejszymi wersjami
- [ ] Skonfiguruj system cache'owania (W3 Total Cache, WP Rocket są dobrymi opcjami) albo aktywuj cache po stronie hostingu
- [ ] Upewnij się, że witryna posiada aktualny certyfikat SSL i wykorzystuje protokół HTTP/2
- [ ] Aktywuj kompresję Gzip lub Brotli w konfiguracji serwera lub za pomocą odpowiedniej wtyczki
- [ ] Wykonaj generalne porządki w bazie danych - wyczyść stare rewizje artykułów, komentarze spam oraz pliki tymczasowe
- [ ] Zminimalizuj pliki CSS i JavaScript wykorzystując wtyczki lub narzędzia deweloperskie
- [ ] Ustaw regularne kopie zapasowe wykonywane przynajmniej cotygodniowo
- [ ] Wdróż monitoring dostępności strony przy użyciu rozwiązań takich jak UptimeRobot
- [ ] Przeprowadź ponowny test wydajności w Google PageSpeed Insights i porównaj otrzymane rezultaty z wartościami początkowej
