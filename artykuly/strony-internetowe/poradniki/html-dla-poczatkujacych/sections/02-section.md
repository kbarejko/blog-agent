## Czym jest HTML i jak działa - podstawy bez technicznego żargonu

### Co oznacza skrót HTML

HTML to HyperText Markup Language – język znaczników hipertekstu. Brzmi skomplikowanie? W praktyce to po prostu sposób na opisanie, co znajduje się na stronie internetowej.

Historia HTML sięga 1990 roku, kiedy Tim Berners-Lee stworzył pierwszy język do publikowania dokumentów w internecie. Od prostego systemu łączenia tekstów ewoluował do narzędzia budowy kompleksowych stron biznesowych. Każda wersja HTML dodawała nowe możliwości – od podstawowych linków do zaawansowanych formularzy kontaktowych.

HTML działa w zespole z dwoma innymi technologiami. CSS odpowiada za wygląd – kolory, czcionki, rozmieszczenie elementów. JavaScript dodaje interaktywność – animacje, formularze, reakcje na kliknięcia. HTML pozostaje fundamentem – określa, co się na stronie znajduje.

Przeglądarka interpretuje kod HTML jak przepis kulinarny. Czyta instrukcje krok po kroku i buduje stronę zgodnie z opisem. Tag `<h1>` oznacza "tu jest główny nagłówek", `<p>` znaczy "to akapit tekstu". Przeglądarka wie, jak każdy element wyświetlić użytkownikowi.

### Struktura dokumentu HTML

Wyobraź sobie HTML jak szkielet budynku. Fundament, ściany nośne, podłogi – wszystko ma swoje miejsce i funkcję. Bez solidnej struktury nawet najpiękniejsze wykończenie się zawali.

Znaczniki (tagi) to elementy konstrukcyjne HTML-a. Każdy tag ma konkretne zadanie – `<header>` tworzy nagłówek strony, `<main>` oznacza główną treść, `<footer>` to stopka z informacjami kontaktowymi. To jak etykiety na segregatorach – pomagają zorganizować zawartość.

Kluczowa różnica: HTML definiuje strukturę i znaczenie, nie wygląd. Tag `<h1>` mówi "to najważniejszy nagłówek", ale nie określa jego koloru czy rozmiaru. To zadanie CSS-a. Dzięki temu ta sama treść może wyglądać inaczej na telefonie i komputerze, zachowując logiczną strukturę.

Właściciel restauracji dodający menu do strony używa tagów `<ul>` dla listy dań i `<li>` dla każdej pozycji. Struktura pozostaje czytelna niezależnie od designu.
