## Checklist: HTML dla przedsiębiorców - Wdrożenie w praktyce

- [ ] Pobierz darmowy edytor kodu (polecam Visual Studio Code lub prosty Notepad++) i stwórz swój pierwszy plik HTML - to podstawa do dalszej pracy
- [ ] Sprawdź meta tagi na każdej podstronie - title powinien mieć maksymalnie 60 znaków, a description około 150-160 znaków z kluczowymi hasłami
- [ ] Przeanalizuj wszystkie zdjęcia i grafiki - każda powinna mieć opisowy tekst alt zawierający nazwę Twojej firmy i główne słowa kluczowe dla tej strony
- [ ] Zweryfikuj hierarchię nagłówków H1-H6 na każdej podstronie - jeden H1 z najważniejszym hasłem, potem H2, H3 według logicznej struktury treści
- [ ] Wdróż dane strukturalne Schema.org dla swojej działalności - dodaj podstawowe informacje jak adres siedziby, numer telefonu kontaktowego i godziny pracy
- [ ] Przetestuj działanie formularza kontaktowego - sprawdź czy każde pole ma czytelną etykietę i czy system waliduje wprowadzone dane przed wysłaniem
- [ ] Zweryfikuj responsywność witryny na smartfonach i tabletach korzystając z narzędzi deweloperskich przeglądarki (klawisz F12)
- [ ] Dodaj przemyślane linki wewnętrzne łączące powiązane treści - na przykład ze strony usług do konkretnych produktów czy z artykułów do oferty
- [ ] Uruchom walidator HTML W3C dla swojej strony i systematycznie popraw wszystkie wykryte błędy w kodzie
- [ ] Skonfiguruj podstawowe narzędzia monitorowania - Google Analytics do śledzenia ruchu i Google Search Console do analizy pozycjonowania
- [ ] Utwórz kopię zapasową obecnej wersji strony zanim zaczniesz wprowadzać jakiekolwiek modyfikacje w kodzie
- [ ] Zaplanuj cykliczne przeglądy technicznej strony co 3 miesiące i dodaj przypomnienie do swojego kalendarza biznesowego
