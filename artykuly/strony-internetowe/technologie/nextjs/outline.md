## 2. Next.js App Router - przełom w budowaniu nowoczesnych stron
(~350 słów)

### Zalety nowej architektury
- Server Components vs Client Components - kiedy używać których
- Streaming i Suspense dla lepszego UX
- Automatyczne code splitting i lazy loading
- Nested layouts i template system

### Wpływ na SEO i wydajność
- Renderowanie po stronie serwera (SSR) out-of-the-box
- Static Site Generation (SSG) dla stron katalogowych
- Incremental Static Regeneration (ISR) dla dynamicznych treści
